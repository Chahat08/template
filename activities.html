Monitoring Active Fire Locations (AFLs)

Gurjeetpal Bawa, Samandeep Kaur, Abheer Narula, Kapish Mittal, Harleen Singh, Yuvraj Mann

Developing website and an android application to locate fire locations by using mobile app under the project “Multi modal framework for monitoring active fire location (AFL) and precision in allied agricultural activities using communication technologies”.

Social distance Violation

Anup Ahlawat, Kartik Pathak, Kanav Sharma, Kabir

Application to measure the social distance between humans over drone data and detect any covid-19 violation for social distancing. Its carry out as:

  • Human detection over data.
  • Calculating coordinates and forming safe zone radius.
  • Radius overlapping of 3 feet (diameter 6 feet) considered as violation.

Drone Live Video Transfer System

Rudra

This implement the live video transfer application from drone to a base server and then stream the output with the help of broadcasting software to multiple users. The system comprises of three different components:

  • Raspberry Pi with Gstreamer (Hardware+Software)
  • Gstreamer on Base Server (Software)
  • Broadcasting Server
These three components comprise the system to initiate the transfer, capture and distribute the live feed from the drone.

SIH BASTET

Rudra

Application to enhance the present features of Telecom infrastructure like Open CellID Database.

  • Mapped the telecom towers in QGIS with their network coverage around their placement point
  • To find out the network devoid areas with the help of this data
  • Reduce cost by efficient network planning
  • Bringing high speed internet to rural areas
These three components comprise the system to initiate the transfer, capture and distribute the live feed from the drone.

Covmap Web-GIS

Rudra

Web based GIS for the Coronavirus Pandemic to create a visualization platform with not just numbers and charts, but a real-time application that would help everyone view the covid spread in their area, view hospitals. more features such as Drive through and vaccination centre as they would come. The approach followed is:

  • Implement a geospatial Restful-API with post-gis database
  • Visualise the data on a map canvas with real time monitoring from a public or private api for covid cases
